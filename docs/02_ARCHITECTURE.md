# 🏗️ Architecture Documentation

## 🛠️ Tech Stack
- **Runtime:** Node.js (v20+)
- **Framework:** NestJS (v11+)
- **ORM:** Prisma (v7.8) con Driver Adapters.
- **DB:** PostgreSQL (v16) hospedado en Supabase / Docker local.
- **Package Manager:** PNPM (Monorepo).

## 🗺️ Estructura del Monorepo
- /apps/api: Núcleo del Backend (NestJS).
- /docs: Documentación técnica y de producto.
- /packages: (TBD) Shared types o utilitarios.
